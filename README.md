# dad-list-generator
DAD (daily action dude) list generator written in Kotlin.

## Example result

```text
Monday: Dude 4
Tuesday: Dude 7
Wednesday: Dude 3
Thursday: Dude 5
Friday: Dude 6

Generated on 01.01.1970 01:00:00
```
